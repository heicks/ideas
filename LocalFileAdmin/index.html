# **`书籍命名规定`**： > [中文书译名[ OR 中文书译名2], 英文原书名] [.nth (n为版本号)] [英(=英文版),中(=中文版)] [作者[,作者,...] [译者,[译者,..]]] [[原出版社, 本书第三方出版社]] [页数] [扫(=扫描版),文=(文字版) [实体版物价范围]] # 注释： >> 【1】： `app_PATH` 当前路径起始部分为只读属性
